# Sound Assets

This folder should contain 8-bit/chiptune sound effects for the pixel-art aesthetic.

## Required Files

| File | Purpose | Recommended Duration |
|------|---------|---------------------|
| `tap.wav` | Button tap feedback | 50-100ms |
| `start.wav` | Timer/session start | 200-400ms |
| `complete.wav` | Session complete (success chime) | 500-800ms |
| `purchase.wav` | Shop item purchased | 200-400ms |
| `quest_complete.wav` | Quest completed | 400-600ms |
| `error.wav` | Error occurred (low thud) | 150-300ms |

## Style Guidelines

- Use 8-bit/chiptune style to match pixel-art aesthetic
- Keep files small (< 50KB each)
- Use WAV format for best compatibility
- Sounds should be satisfying but not intrusive

## Free Resources

- https://sfxr.me/ - Generate 8-bit sounds
- https://www.bfxr.net/ - Similar generator with more options
- https://freesound.org/ - Search for "8-bit" or "chiptune"
